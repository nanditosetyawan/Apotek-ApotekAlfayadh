# Sistem Informasi Apotek Alfayadh Farma

Apotek Alfayadh Farma merupakan sistem informasi yang dirancang untuk mendukung operasional apotek secara terintegrasi, meliputi pengelolaan transaksi penjualan, manajemen stok obat, serta pelaporan keuangan dan persediaan. Sistem ini dikembangkan dengan pendekatan berbasis peran (*role-based system*) untuk memastikan setiap pengguna memiliki akses sesuai tanggung jawabnya.

## Role dan Fungsi

- Kasir  
  Bertanggung jawab dalam melakukan transaksi penjualan obat, menghitung total pembayaran secara otomatis, serta mencatat dan menghasilkan laporan pemasukan.

- Manager Gudang  
  Bertugas mengelola data obat, menambahkan stok, memperbarui detail obat, serta membuat laporan obat masuk.

- Manager Cabang  
  Berperan dalam memantau laporan pemasukan dan laporan stok obat untuk mendukung pengambilan keputusan.
